# 🎵 MusicConnect Korea (가제)
> 당신의 인생곡을 소개하고, 한국의 숨은 명곡을 함께 찾아가는 음악 공유 플랫폼

## 1. 프로젝트 개요
사용자들이 자신이 좋아하는 한국 음악(K-Pop, 인디, 시티팝 등)을 추천하고, 다른 사용자들과 음악적 취향을 나누는 커뮤니티형 웹서비스입니다.

## 2. 주요 기능
### 👤 사용자 기능
- **로그인/회원가입**: 소셜 로그인(카카오, 네이버) 연동으로 접근성 강화
- **마이페이지**: 내가 추천한 곡 목록 및 '좋아요'한 플레이리스트 관리

### 🎶 음악 공유 및 추천
- **포스트 작성**: 앨범 아트, 곡 제목, 아티스트, 추천 이유 작성 및 YouTube 링크 삽입
- **카테고리 분류**: 장르별(발라드, 힙합 등), 무드별(비오는 날, 드라이브 등) 필터링
- **실시간 인기글**: 추천(좋아요) 수에 따른 인기 음악 랭킹

### 💬 커뮤니티
- **댓글 및 답글**: 추천받은 음악에 대한 소감 공유
- **팔로우 시스템**: 취향이 비슷한 유저 팔로우

## 3. 기술 스택 (추천)
- **Frontend**: React.js 또는 Next.js (SEO 최적화에 유리)
- **Backend**: Node.js (Express) 또는 Python (Django/FastAPI)
- **Database**: PostgreSQL 또는 MySQL
- **Storage**: AWS S3 (이미지 업로드용)
- **API**: YouTube Data API (음악 미리보기 및 검색)

## 4. 작업 순서 (Roadmap)
1.  **기획 및 와이어프레임**: UI/UX 설계 (Figma 활용)
2.  **데이터베이스 모델링**: 회원, 게시물, 댓글 구조 설계
3.  **백엔드 API 개발**: CRUD(생성, 조회, 수정, 삭제) 기능 구현
4.  **프론트엔드 UI 개발**: 반응형 웹 디자인 적용
5.  **기능 통합 및 테스트**: API 연동 및 버그 수정
6.  **배포**: Vercel 또는 AWS를 통한 서비스 런칭
